# Collaborative Drawing App (Next.js(14) , express )

## Installation

Use the package manager [npm](https://nodejs.org/en)

```bash
cd server
npm i
npm run server
cd ..
cd client
npm i
npm run server
```

## usage

Just open 2 tabs with localhost:3000.
